# Catálogo de Libros - Proyecto Java

Este proyecto es un **Catálogo de Libros** desarrollado en **Java** que interactúa con los usuarios a través de la consola. Los usuarios pueden buscar libros, ver detalles de libros y almacenarlos en una base de datos. El catálogo consume información de libros desde una **API pública** y proporciona varias opciones de interacción.

## Funcionalidades

1. **Buscar libros por título**: Los usuarios pueden buscar libros ingresando un título.
2. **Buscar libros por autor**: Los usuarios pueden buscar libros por el nombre del autor.
3. **Mostrar todos los libros**: Muestra todos los libros disponibles en la base de datos.
4. **Mostrar libros por categoría**: Los usuarios pueden filtrar los libros por género o categoría.
5. **Salir**: Opción para salir de la aplicación.

## Tecnologías Usadas

- **Java**: Lenguaje principal utilizado para el desarrollo del proyecto.
- **MySQL/PostgreSQL**: Base de datos para almacenar información sobre los libros.
- **Gson**: Biblioteca para manejar la conversión de respuestas JSON de la API.
